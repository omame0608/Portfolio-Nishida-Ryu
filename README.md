# Portfolio-Nishida-Ryu
西田琉の制作物のコードです。ゲーム以外の制作物も含みます。  
一部制作物についてはプレイ動画や説明資料、実行ファイル等も含みます。

最も自信のある作品は「Perfect Back Parker」です。  
ロジックと見た目を可能な限り分離し、コアルールはpureC#で構成されています。

2025/11/8追記  
最新作「Perfect Back Parker Rev!」を追加しました。

是非ご一読いただけますと幸いです。

2025/11/24追記  
最新作「Supernova Trolley」を追加しました。  
VRという複雑なプラットフォームでの開発だったこともあり、これまでの「ロジックと見た目の分離」によるメリットの享受を強く感じた、かなり印象深い制作となりました。  
制作時には初日にロジックのみpureC#で書き切り、VR上での見た目作成・MonoBehaviourレイヤーからロジックレイヤーへのユースケース呼び出し・演出等を後から付け加えていくイメージで、制作時間約2.5日という迅速な開発が行えました。
