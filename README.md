# Portfolio-Nishida-Ryu
西田琉の制作物のコードです。ゲーム以外の制作物も含みます。  
一部制作物についてはプレイ動画や説明資料、実行ファイル等も含みます。

**最も自信のある作品は「Perfect Back Parker Rev!」です。**  
レイヤーを強く意識したアーキテクチャで構築しており、大きく「ロジックレイヤー」「ビューレイヤー」の2層から成っています。  
なかでも「ロジックレイヤー」については上手く丁寧に組み立てることが出来たなと思っています。  
是非ご一読いただけますと幸いです。　　

2025/11/8追記  
最新作「Perfect Back Parker Rev!」を追加しました。  
Indie Games Contest学生選手権2025ベスト12に選出いただいた前作「Perfect Back Parker」をはるかに上回るものになっていると自負しています。  
現在この作品で「ゲームクリエイター甲子園2025」に挑戦中です。

2025/11/24追記  
最新作「Supernova Trolley」を追加しました。  
VRという複雑なプラットフォームでの開発だったこともあり、これまでの「ロジックと見た目の分離」によるメリットの享受を強く感じた、かなり印象深い制作となりました。  
制作時には初日にロジックのみpureC#で書き切り、VR上での見た目作成・MonoBehaviourレイヤーからロジックレイヤーへのユースケース呼び出し・演出等を後から付け加えていくイメージで、制作時間約2.5日という迅速な開発が行えました。

2025/12/11追記  
過去作「Talk & Draw」を最新版にアップデートしました。  
サーバを利用したwebアプリケーションで、チャット・掲示板機能や少し特殊なガチャ要素をメインに、ユーザ間でコミュニケーションを楽しむことが出来ます。  
複数人での開発であり、私はクライアントサイドのみ担当しています。  
サーバサイドについては別途リポジトリ情報を情報.txtに記載しています。
